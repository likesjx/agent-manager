# Agent Manager

Repo-embeddable, provider-agnostic agent operations module for:

- Onboarding
- Handoffs
- Team workflows
- Code-management governance
- Work intake normalization (ADO, iTrack)

## v1 agent clients

- GitHub Copilot
- Claude Code
- Gemini CLI / Antigravity
- Windsurf
- Cursor
- OpenCode
- AskArchitect

## Quick start

1. Define profile in `profiles/`.
2. Register provider adapters in `providers/`.
3. Run validation: `scripts/validate.sh`.
